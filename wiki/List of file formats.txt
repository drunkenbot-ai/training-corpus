( Discuss ) ( March 2026 ) The typical file name extension used for a format is included in parentheses if it differs from the identifier, ignoring case.

In the case of cartridge-based software, if the platform specific extension is not used then filename extensions ".rom" or ".bin" are usually used to clarify that the file contains a copy of a content of a ROM.
